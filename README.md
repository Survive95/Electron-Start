# Electron Start

## To Use

```bash
# Clone this repository
git clone https://github.com/Survive95/Electron-Start.git

# Go into the repository
cd Electron-Start

# Install dependencies
npm install
# or
yarn install

# Run the app
npm start
# or
yarn start
```
